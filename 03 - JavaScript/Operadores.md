# Tabela de Operadores em JavaScript

| **Operador** | **Operação**          | **Exemplo**  |
|--------------|-----------------------|--------------|
| `>`          | Maior que             | `(a > b)`    |
| `<`          | Menor que             | `(a < b)`    |
| `>=`         | Maior ou igual a      | `(a >= b)`   |
| `<=`         | Menor ou igual a      | `(a <= b)`   |
| `==`         | Igual a               | `(a == b)`   |
| `!=`         | Diferente de          | `(a != b)`   |
| `===`        | Idêntico a            | `(a === b)`  |
| `!==`        | Não idêntico a        | `(a !== b)`  |
| `&&`         | E/and                | `(a && b)`   |
| `||`         | Ou/or                | `(a || b)`   |
